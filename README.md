# Ecoji - Sustainable Living Platform

Ecoji is a premium, modern e-commerce platform dedicated to sustainable, eco-friendly products. Crafted with a beautiful, cinematic 3D storytelling experience and highly optimized for performance and accessibility.

## 🚀 Features

- **Cinematic 3D Ecosystem:** Fully procedural React Three Fiber bamboo forest with real-time mouse wind interactions and GSAP scroll sequences.
- **Premium UI/UX:** Glassmorphic elements, bespoke cursors, and extremely smooth Lenis scrolling.
- **Responsive Design:** Completely optimized for both desktop and mobile experiences with dynamic DPR scaling.
- **Modern Tech Stack:** 
  - [React 19](https://react.dev/)
  - [Vite](https://vitejs.dev/)
  - [TailwindCSS v3](https://tailwindcss.com/)
  - [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)
  - [GSAP](https://gsap.com/)
  - [Zustand](https://zustand-demo.pmnd.rs/)

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd ecoji
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

## 🏗️ Project Architecture

- `src/components/3d/` - Modular React Three Fiber components for the procedural ecosystem.
- `src/components/` - Global UI components (Navbar, Footer, ProductCards).
- `src/pages/` - Main page routes (Home, Products, Sustainability).
- `src/store/` - Zustand global state management.
- `src/config/` - Static configurations and dummy product data.
